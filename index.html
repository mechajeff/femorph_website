Born at AFRL, Hardened at Pratt & Whitney, Ready for Production.

FEMORPH was conceived and developed at the Air Force Research Laboratory (AFRL) to address the challenge of modeling as-manufactured geometries with simulation-grade accuracy.

The technology has been industry-hardened in collaboration with Pratt & Whitney, where it has demonstrated robust performance across industrial-scale use cases in digital twin creation, mesh adaptation, and optimization workflows.

FEMORPH is now ready for production deployment in aerospace, energy, and advanced manufacturing environments.

Integrated with F-22 Maintenance, Repair, Overhaul Program

FEMORPH is in operational use at Air Force Sustainment Center at Tinker Air Force Base. It has been integral to the repair and maintenance of the Air Forces most advanced fighter aircraft.

Basic FEMORPH Target Surface Process

1. Collect Geometry Data

Geometry data is collected using high-precision tools such as coordinate measuring machines (CMMs) and optical 3D scanners. These produce detailed point clouds or surface meshes for each component.

2. Import into FEMORPH

The measurement data is imported into FEMORPH, where it can be analyzed alongside the baseline simulation model.

3. Align and Compare

FEMORPH aligns the measured geometry and baseline model, producing a detailed deviation map that quantifies local shape differences.

4. Morph the Mesh

Robust mesh morphing transfers the measured geometric deviations to the simulation model, eliminating the need for time-consuming remeshing.

5. Save the script for automated morphing next data set

The steps performed in the GUI are implementing python function calls. These are recorded into a script for reuse.
